# 第一节 区块链价值与原理
1、主要讲了web2数据比较集中，而web3数据是分散的，是在一个分布式网络上的。数据修改起来不那么容易，用户是拥有对数据的所有权
2、区块链通过密码学让数据控制权回归用户、通过去中心化网络来防止组织的作恶。基本原理，共识机制用来协调去中心化网络中的节点运行。
3、BTC 共识：谁先完成 POW 难度计算，谁获奖励，同时使用 Hash区块链式结构让验证简单。


# 第二节 以太坊核心概念及第一个合约
1、以太坊改进：植入虚拟机（EVM）用来执行程序

3、程序在 EVM 上执行时，根据每个指令扣费（gas）
